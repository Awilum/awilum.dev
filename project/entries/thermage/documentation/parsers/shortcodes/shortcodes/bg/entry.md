#### Examples

Set named background color style 
```
[bg=red]Stay RAD[/bg]
```

Set hex background color style 
```
[bg=#FF0000]Stay RAD[/bg]
```

Set rgb background color style 
```
[bg="rgb(255, 0, 0)"]Stay RAD[/bg]
```
