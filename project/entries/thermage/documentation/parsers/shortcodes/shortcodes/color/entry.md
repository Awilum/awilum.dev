#### Examples

Set named text color style 
```
[color=red]Stay RAD[/color]
```

Set hex text color style 
```
[color=#FF0000]Stay RAD[/color]
```

Set rgb text color style 
```
[color="rgb(255, 0, 0)"]Stay RAD[/color]
```
